# Fake-News-Classifier-NLP-
Machine learning project that detects fake news using NLP techniques, TF-IDF vectorization, and Logistic Regression in Python.
# Fake News Detection using Machine Learning

This project detects fake news using NLP and machine learning.

Tools Used:
Python
Scikit-learn
TF-IDF Vectorization
Pandas
NumPy

Steps:
1. Data preprocessing
2. Text cleaning
3. Feature extraction using TF-IDF
4. Model training using Logistic Regression
5. Model evaluation

Result:
Achieved good accuracy in classifying fake and real news.
